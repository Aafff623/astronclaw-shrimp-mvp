## 录制建议

- **分辨率**：1920×1080 起步
- **工具**：OBS Studio / ScreenToGif / QuickTime
- **配音**：可选，但建议加字幕
- **节奏**：开头 5 秒抓眼球，结尾 5 秒留 GitHub 仓库地址
- **备份**：本地保留原始视频文件，避免平台失效
